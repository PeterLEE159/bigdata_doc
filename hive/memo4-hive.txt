Hive
    - Apache 하둡 프로젝트의 하위 프로젝트다.
    - 데이터 웨어하우스 사용을 위한 SQL 제공
    - Hive는 SQL에 익숙한 사용자와 ad hoc쿼리를 필요로 하는 사용자들을 위해 개발되었다.
    - 하둡의 대용량 데이터 분석을 위해 HiveQL구문이라는 SQL과 유사한 언어를 사용해 데이터를 처리한다.
    - Hive는 구조화된 데이터를 쿼리하고, 시스템을 관리하기 위한 기능을 제공한다.
    - Hive는 구조화된 데이터를 대상으로 복잡한 MapReduce 프로그램 대신 HiveQL을 사용해서 쉽게 데이터를 처리한다.
    - Hive는 관계형 데이터베이스와 유사한 개념을 사용한다.
    - 하둡은 파일을 직접 처리하지만, Hive는 디렉토리 구조로 분산된 데이터를 사용한다.
    
    - 주요 컴포넌트
        - metastore
            - 데이터 스키마 정보(메타정보)를 저장하는 저장소다.
            - JDBC를 지원하는 모든 데이터베이스에 메타스토어를 구축할 수 있다.
        - HiveQL
            - 사용자가 입력한 HiveQL구문을 분석한다.
            - 분석된 HiveSQ구문에 대한 수행계획을 작성한다.
            - 수행계획에 맞는 MapReduce 코드를 생성한다.
            - Parser, Planner, Optimizer, Execution으로 구성되어 있다.
    - 설치
        - hive 설치파일 다운로드
            - http://hive.apache.org/
        - 환경변수에 HIVE_HOME 등록
            - export HIVE_HOME={{pwd}}
        - PATH에 $HIVE_HOME/bin 등록
            - export PATH=$HIVE_HOME/bin:$PATH
    - 실행
        - 환경변수에 HADOOP_HOME 등록
            - export HADOOP_HOME=<hadoop-install-dir>
        - HDFS에 /tmp, /user/hive/warehouse 폴더 생성 및 쓰기권한 부여
            - $HADOOP_HOME/bin/hadoop fs -mkdir       /tmp
            - $HADOOP_HOME/bin/hadoop fs -mkdir       /user/hive/warehouse
            - $HADOOP_HOME/bin/hadoop fs -chmod g+w   /tmp
            - $HADOOP_HOME/bin/hadoop fs -chmod g+w   /user/hive/warehouse
